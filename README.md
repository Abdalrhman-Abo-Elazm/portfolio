🎯 Overview  
This project is a professional personal portfolio created by **Abdalrhman Aboalzm** to display his web development skills, educational background, and contact information. It is structured as a single-page website with smooth scrolling, interactive elements, and a strong visual layout.

🛠 Tech Stack  
- **HTML5** – Semantic and accessible markup  
- **CSS3** – Custom styling with Flexbox & responsive design  
- **JavaScript (Vanilla)** – Smooth UI interactions  
- **FontAwesome** – Iconography via CDN  
- **Netlify** – Hosting and deployment

🚀 Features  
✅ Clean and responsive layout  
🎨 Modern visual design with smooth scrolling  
🧑‍💻 Dedicated sections: About, Skills, Projects, Contact  
🖼️ Gallery of screenshots from actual work  
📄 Embedded downloadable CV  
📱 Fully responsive on all devices  
🌐 Hosted on Netlify for global access  

📌 Planned Enhancements

📊 Add analytics for traffic monitoring

🔧 Installation  
Clone the repository and open in your browser:

```bash
git clone https://github.com/Abdalrhman-Abo-Elazm/portfolio.git
cd portfolio
```

Then open `index.html` in any browser.
